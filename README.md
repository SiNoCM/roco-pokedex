# 《洛克王国·世界》精灵图鉴

基于 React + TypeScript + Vite 构建的洛克王国世界精灵图鉴 Web 应用，收录 406 只精灵完整数据。

## 在线访问

[https://sinocm.github.io/roco-pokedex/](https://sinocm.github.io/roco-pokedex/)

## 功能特点

- **精灵图鉴**：收录 406 只精灵，支持按属性筛选、搜索、排序
- **属性克制系统**：18x18 完整属性克制表，支持双属性叠加计算（4x/2x/1x/0.5x/0.25x/免疫）
- **游戏图标整合**：598 个游戏原始图标（469 技能 + 129 特性），18 种属性图标
- **赛季异色系统**：S1「暗夜拾光」49 只 + S2「狂欢怪谈」26 只 + S3「铅字幻梦」53 只，共 128 只异色精灵
- **进化链导航**：详情页底部展示同进化链精灵，支持一键跳转
- **精灵详情**：属性、种族值雷达图、特性、技能、变体形态等完整信息
- **变体形态**：首领形态、地区形态、皮肤形态整合展示
- **响应式设计**：适配电脑端与手机端
- **PWA 离线支持**：Service Worker 缓存，图片总大小仅 13.9MB

## 数据说明

- 精灵数据来源于游戏数据库，共 406 只精灵
- 赛季异色共 128 条记录（S1=49, S2=26, S3=53）
- 技能数据按类型分为：普通技能、血脉技能、技能石
- 属性克制数据经游侠网/17173 攻略验证
- 图片资源来自 BWiki（哔哩哔哩洛克王国百科）
- 图片经 WebP 压缩优化，总计 667 张仅 13.9MB
