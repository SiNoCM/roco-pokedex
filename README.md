# 洛克王国精灵图鉴

基于 React + TypeScript + Vite 构建的洛克王国精灵图鉴 Web 应用。

## 在线访问

🌐 [https://sinocm.github.io/roco-pokedex/](https://sinocm.github.io/roco-pokedex/)

## 功能特点

- **精灵图鉴**：收录 347 只精灵，支持按属性筛选、搜索、排序
- **精灵详情**：查看精灵属性、种族值、特性、技能等完整信息
- **技能分类**：技能按「本身技能」「血脉技能」「技能石技能」三类展示
- **变体形态**：同一精灵的不同形态（首领形态、地区形态、皮肤形态）整合展示，标注获取方式
- **无限滚动**：向下滚动自动加载更多精灵
- **响应式设计**：适配电脑端与手机端分辨率

## 数据说明

- 精灵数据来源于 SQLite 数据库，共 465 条记录
- 去重后 347 只主精灵，163 个变体形态
- 变体类型包括：首领形态、地区形态、皮肤形态
- 技能数据按类型分为：普通技能、血脉技能、技能石
- 图片资源共 465 张（含主精灵图片和变体皮肤图片）

## 技术栈

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4
- Lucide React 图标库
