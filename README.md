# 洛克王国精灵图鉴

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Deployed-brightgreen)](https://sinocm.github.io/roco-pokedex/)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

> 收录347只精灵（含118个变体形态）完整数据的在线图鉴

🔗 **在线访问**: https://sinocm.github.io/roco-pokedex/

## 功能特点

- 📚 **完整收录** - 347只精灵 + 118个变体形态
- 🔄 **变体展示** - 同一编号下不同形态（皮肤、地区形态、首领形态）整合展示
- 🏷️ **获取方式** - 每个变体标注形态类型和来源
- 🔍 **智能搜索** - 支持名称、编号、属性搜索
- 🎨 **属性筛选** - 18种属性快速筛选
- ⚔️ **技能分类** - 区分本身技能、血脉技能、技能石技能
- 📱 **响应式设计** - 适配手机、平板、电脑
- ⚡ **无限滚动** - 流畅的加载体验

## 数据说明

- **精灵总数**: 347只（去重后）
- **变体形态**: 118个（皮肤、地区形态、首领形态等）
- **技能数据**: 包含普通技能、血脉技能、技能石技能
- **数据来源**: 《洛克王国：世界》手游

## 技术栈

- Tailwind CSS

- GitHub Pages
